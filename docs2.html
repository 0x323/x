1. Overview

CheckInPoint is a gamified token system on Sepolia Base. Users earn Points through daily check-ins, referrals, and betting.

  • Daily Check-In: Earn 1 Point per day.
  • Referral Program: Earn 5 Points per bet made by referees.
  • Betting: Place bets, win rewards based on number prefix matches.

Token Point has 18 decimals. Transfers are disabled to control distribution.

3. Data Structures

Bet

Stores details about a user's bet.
struct Bet {
address user;
string number;
uint256 cost;
uint256 reward;
uint256 timestamp;

}

ReferralInfo

Tracks referral relationships and balances.
struct ReferralInfo {
string referralId;
string referrerId;
address referrer;
address[] referees;
uint256 referralBalance;

}

CheckInLog

Logs daily check-ins.
struct CheckInLog {
address user;
uint256 dayUTC;

}
