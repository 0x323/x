Pendahuluan

CheckInPoint adalah aplikasi berbasis blockchain (DApp) di jaringan Sepolia Base, sebuah jaringan pengujian untuk Ethereum. Dengan DApp ini, kamu bisa:

  • Check-in harian: Dapatkan 1 Poin setiap hari hanya dengan mengklik tombol.
  • Program referral: Undang teman dan dapatkan 5 Poin saat mereka memasang taruhan.
  • Betting: Pasang taruhan dengan nomor pilihanmu untuk memenangkan hadiah hingga 10 Poin atau lebih.
  • Poin: Token yang digunakan dalam DApp, tidak bisa ditransfer langsung ke orang lain.

Tutorial ini akan memandu kamu melalui langkah-langkah untuk mulai menggunakan DApp, dari menyiapkan dompet hingga berinteraksi dengan semua fitur. Kamu hanya perlu browser web (seperti Chrome) dan dompet kripto MetaMask.

Prasyarat

Sebelum mulai, pastikan kamu memiliki:

  1. Perangkat dengan Browser: Komputer atau ponsel dengan browser seperti Google Chrome, Firefox, atau Safari.
  2. MetaMask Terinstal:
    • Komputer: Tambahkan ekstensi MetaMask dari metamask.io.
    • Ponsel: Instal aplikasi MetaMask dari App Store (iOS) atau Play Store (Android).
  3. ETH Sepolia: Sedikit ETH di jaringan Sepolia untuk biaya transaksi (dapatkan gratis dari Base Sepolia Faucet).
  4. Akses ke DApp: Buka DApp CheckInPoint di https://checkinpoint-dapp.com (ganti dengan URL sebenarnya).

Langkah 1: Menyiapkan MetaMask untuk Sepolia Base

  1. Buka MetaMask:
    • Klik ikon MetaMask di browser (pojok kanan atas) atau buka aplikasi di ponsel.
    • Masuk menggunakan kata sandi.
  2. Tambahkan Jaringan Sepolia Base:
    • Klik nama jaringan (misalnya, “Ethereum Mainnet”) di atas.
    • Pilih Tambahkan Jaringan.
    • Masukkan detail:
      ParameterNilai
      Nama JaringanSepolia Base
      URL RPC Baruhttps://sepolia.base.org
      Chain ID84532
      Simbol Mata UangETH
      Block Explorer URLhttps://sepolia.basescan.org
    • Klik Simpan.
    • Pastikan jaringan beralih ke Sepolia Base.
  3. Dapatkan ETH Sepolia:
    • Salin alamat dompet (klik alamat di MetaMask, dimulai dengan 0x...).
    • Kunjungi Base Sepolia Faucet.
    • Tempel alamat, selesaikan verifikasi, dan kirim permintaan.
    • Tunggu ETH masuk (biasanya 0.1-0.5 ETH).
  4. Verifikasi Saldo: Pastikan saldo ETH Sepolia lebih dari 0.

Langkah 2: Mengakses DApp CheckInPoint

  1. Buka Situs DApp:
    • Kunjungi https://checkinpoint-dapp.com.
    • Pastikan URL benar untuk menghindari situs phishing.
  2. Hubungkan MetaMask:
    • Cari tombol Connect Wallet atau Hubungkan Dompet.
    • Klik tombol, lalu pilih akun di MetaMask dan klik Connect.
    • Jika berhasil, DApp akan menampilkan alamat dompet kamu.
  3. Verifikasi Koneksi:
    • Pastikan jaringan adalah Sepolia Base.
    • Jika gagal, refresh halaman atau login ulang ke MetaMask.

Langkah 3: Melakukan Check-In Harian

Dapatkan 1 Poin setiap hari dengan check-in.

  1. Temukan Tombol Check-In: Cari tombol Check-In atau ikon kalender di halaman utama.
  2. Lakukan Check-In:
    • Klik Check-In.
    • Konfirmasi transaksi di MetaMask (klik Confirm).
    • Tunggu 5-30 detik hingga muncul pesan “Check-in berhasil!”.
  3. Cek Saldo Poin: Lihat bagian Balance untuk memastikan saldo bertambah 1 Poin.
  4. Penanganan Error:
    • “You have already checked in today”: Coba lagi besok (pukul 07:00 WIB).
    • Transaksi gagal: Periksa saldo ETH Sepolia.
  5. Tips: Check-in setiap hari untuk mengumpulkan Poin.

Langkah 4: Menggunakan Program Referral

Undang teman dan dapatkan 5 Poin saat mereka memasang taruhan.

  1. Atur ID Referral:
    • Pergi ke bagian Referral.
    • Masukkan ID unik (misalnya, myid123).
    • Klik Set ID dan konfirmasi di MetaMask.
  2. Bagikan ID Referral:
    • Salin ID dari DApp.
    • Bagikan ke teman (contoh: “Gunakan ID myid123 di CheckInPoint!”).
  3. Teman Bergabung:
    • Minta teman buka DApp, hubungkan MetaMask, dan masukkan ID kamu di bagian Join Referral.
    • Mereka harus konfirmasi transaksi.
  4. Dapatkan Hadiah: Dapatkan 5 Poin saat teman memasang taruhan.
  5. Penanganan Error:
    • “Referral ID already taken”: Pilih ID lain.
    • “You have already set a referral ID”: Gunakan ID yang sudah ada.
    • “Referral ID not valid”: Pastikan ID benar.
  6. Tips: Undang banyak teman untuk lebih banyak Poin.

Langkah 5: Memasang Taruhan (Betting)

Pasang taruhan sekali per hari untuk memenangkan Poin.

  1. Akses Fitur Betting: Cari bagian Bet atau ikon dadu.
  2. Masukkan Detail Taruhan:
    • Nomor: Pilih 1-20 karakter (misalnya, 01234). Bonus 10 Poin untuk nomor mulai “00” atau “01”.
    • Pengali: Pilih 1-10 (misalnya, 2). Biaya = 1 Poin per pengali.
    • Pembayaran: Masukkan jumlah Poin (misalnya, 3).
  3. Konfirmasi Taruhan:
    • Klik Place Bet.
    • Konfirmasi di MetaMask.
    • Tunggu pesan “Taruhan berhasil!”.
  4. Cek Hasil: Lihat Bet History untuk detail taruhan dan periksa saldo.
  5. Penanganan Error:
    • “Insufficient token to place bet”: Dapatkan Poin dari check-in/referral.
    • “You can only place bet once per day”: Coba besok.
    • “Invalid number input”: Gunakan 1-20 karakter.
  6. Tips: Gunakan nomor “00” atau “01” untuk bonus.

Langkah 6: Memeriksa Saldo dan Riwayat

  1. Cek Saldo Poin: Lihat bagian Balance untuk jumlah Poin.
  2. Riwayat Check-In: Periksa Check-In History untuk daftar check-in.
  3. Riwayat Taruhan: Lihat Bet History untuk detail taruhan.
  4. Daftar Referee: Cek My Referrals untuk daftar teman.
  5. Penanganan Error: Refresh halaman jika data tidak muncul.

Langkah 7: Tips untuk Pengguna

  1. Keamanan:
    • Jangan bagikan seed phrase MetaMask.
    • Periksa URL DApp untuk menghindari phishing.
  2. Hemat Biaya Gas: Selalu miliki ETH Sepolia dari faucet.
  3. Maksimalkan Poin:
    • Check-in setiap hari.
    • Undang banyak teman.
    • Eksperimen dengan taruhan.
  4. Pahami Waktu UTC: Check-in dan taruhan direset pukul 07:00 WIB.
  5. Hubungi Dukungan: Hubungi tim CheckInPoint jika ada masalah.
