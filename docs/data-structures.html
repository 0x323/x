1. Overview


The HKBetaTestnet smart contract, deployed on the Sepolia Base network, is a gamification token system designed to drive user engagement through:


  • Daily Check-In: Users earn 1 Point by checking in once per day.

  • Betting Game: Users can place daily bets by earning points that can be accumulated.

  • Referral Program: Users earn 5 Points when their friends have successfully placed a bet.

The contract uses a token called "Lpoint" with 18 decimal places. Direct token transfer is disabled to control transfer.
