CheckInToken Smart Contract Documentation

Comprehensive documentation for the CheckInToken reward-based smart contract

Overview

CheckInToken is a reward-based Ethereum smart contract that allows users to "check-in" and receive points (tokens) daily. It's a basic gamification contract useful for attendance systems, engagement rewards, or participation incentives.

How It Works

  1. User calls checkIn() once per day.
  2. The contract mints a fixed number of tokens (points) to the caller.
  3. Tokens are tracked per user and can be queried using balanceOf().
  4. All check-ins emit CheckIn and TokenMinted events.

Smart Contract Functions

  • checkIn(): Mints points to caller. Emits CheckIn and TokenMinted.
  • balanceOf(address): Returns token balance of the address.
  • totalSupply(): Total minted points in the system.
  • POINTS_PER_CHECKIN(): Constant defining how many points a check-in gives.
  • name(), symbol(), decimals(): ERC-20 metadata methods.

Events

  • CheckIn(address user, uint256 amount, uint256 timestamp): Emitted on each check-in.
  • TokenMinted(address to, uint256 amount): Emitted when tokens are minted.
  • Transfer(address from, address to, uint256 value): Emitted for compatibility but not used in real transfer.

Usage Guide

To use the contract:

  1. Deploy the contract on an EVM-compatible chain (e.g., Ethereum, BNB Chain).
  2. Call checkIn() from a user address (can be called daily).
  3. Use balanceOf(address) to check point balances.
  4. Use totalSupply() to get total points minted.

This contract does not allow transferring tokens between users. It’s purely for point accumulation.

Limitations

  • Tokens are non-transferable.
  • No time restriction logic (e.g., 24-hour limit per check-in) is enforced in this ABI.
  • Does not implement ERC-20 transfer functions beyond dummy interface (pure).

Recommendations

  • Add check-in cooldown tracking with timestamps and mappings.
  • Expand with a leaderboard smart contract to track top users.
  • Optionally make tokens transferable if needed.

Conclusion

CheckInToken is a simple yet effective way to incentivize user interaction via smart contracts. It’s ideal for gamification, attendance logs, or reward systems.
